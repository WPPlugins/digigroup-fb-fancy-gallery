== Description ==
This plugin displays all public albums of a Facebook Page displays your photos using Fancybox 2 Thumbnail helper

= Requirements =
* Wordpress 3.1 or higher
* PHP5 or higher
* jQuery 1.7 or higher

= Language =

* Digigroup Fb Fancy Gallery is currently available in Portuguese

== Installation ==

1. Upload `digigroup-fb-fancy-gallery` to the `/wp-content/plugins/` directory

2. Activate the plugin through the 'Plugins' menu in WordPress

3. Insert `[dgfbfancygallery  fanpage_id="110387505694717"]` in your page or post
